## How to Run
1. Clone the repo:
   ```sh
   git clone https://github.com/your-username/ai-resume-analyzer.git
   ```
2. Navigate to the project folder:
   ```sh
   cd ai-resume-analyzer
   ```
3. Install dependencies:
   ```sh
   npm install
   ```
4. Run the app:
   ```sh
   npm start
   ```
